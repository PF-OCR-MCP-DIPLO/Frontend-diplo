# Build

## Scripts

- `npm run dev`
- `npm run build`
- `npm run preview`
- `npm run typecheck`
- `npm run lint`
- `npm run test`

## Variables Vite

- `VITE_API_BASE_URL`
- `VITE_API_KEY`

## Problemas comunes

- Error de TypeScript: revisar tipos en `src/features/*/types`.
- Build fallido por assets: revisar rutas absolutas servidas por Django.
- Error de red: revisar CORS y API key.
