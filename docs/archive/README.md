# Archivo

Documentación histórica o heredada que se conserva para referencia, pero no es la
fuente canónica.

## Incluye

- `ARCHITECTURE.md`
- `FRONTEND.md`
- `QA_ASSISTANT_E2E.md`
- `TESTING.md`

## Regla

Si un dato cambia de verdad, la actualización debe ir primero a la documentación
canónica y luego reflejarse aquí solo si aporta contexto histórico.
