# Troubleshooting

## API inaccesible

- Causa probable: URL base incorrecta.
- Diagnóstico: abrir la URL configurada en `VITE_API_BASE_URL`.
- Solución: corregir el entorno y reiniciar Vite.

## Errores de autorización

- Causa probable: API key faltante o distinta.
- Diagnóstico: revisar el header `X-API-Key`.
- Solución: sincronizar la clave con el backend.
