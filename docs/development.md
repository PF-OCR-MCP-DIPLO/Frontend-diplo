# Desarrollo

## Propósito

Concentrar scripts y prácticas de trabajo para desarrollo frontend.

## Scripts disponibles

Desde `package.json`:

- `npm run dev`
- `npm run build`
- `npm run preview`
- `npm run lint`
- `npm run typecheck`
- `npm test`
- `npm run test:coverage`
- `npm run e2e`
- `npm run e2e:pw`
- `npm run e2e:pw:ui`

## Flujo recomendado

1. `npm run lint`
2. `npm run typecheck`
3. `npm test`
4. `npm run build`
