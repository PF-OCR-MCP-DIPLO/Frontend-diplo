# ADR 0001: React + Vite + TypeScript

## Estado
Aceptado

## Contexto
La aplicación necesita una SPA rápida, componible y con tipado estático.

## Decisión
Usar React, TypeScript y Vite como stack principal.

## Consecuencias
- Desarrollo rápido y tiempos de build bajos.
- Tipado útil para contratos con la API.
- La estructura de features debe mantenerse ordenada para no dispersar lógica.
